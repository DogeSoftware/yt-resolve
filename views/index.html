yt-resolve api

What's this?

This is a port of yt-resolve. The original project is a CLI app, and this is a web API.

How do I use this?

To get the json data for the first video in your search query, use https://yt-resolve.glitch.me/api/json/YOUR-SEARCH-QUERY. You get a lot of data from this API:

If your app doesn't support JSON for some reason, you can get the url of the video you searched for in plaintext: https://yt-resolve.glitch.me/api/pt/YOUR-SEARCH-QUERY/url.

Unfortunately, you can't get other info (name, duration, channel, views, etc.) using the plain-text api.

There's also a /full endpoint that gives you what it says - all search results, including channels, playlists and more: https://yt-resolve.glitch.me/api/json/YOUR-SEARCH-QUERY/full.

If you want a specific part of the search results (videos, channels or playlists), you can use https://yt-resolve.glitch.me/api/json/YOUR-SEARCH-QUERY/full/videos|playlists|channels. For example, if you want all the playlists, use https://yt-resolve.glitch.me/api/json/YOUR-SEARCH-QUERY/full/playlists.

Warning!

If you're building an app using this API, please send a request to it and wait at least 15 seconds before you start making actual requests. Our hosting provider is "shutting down" this API if there was no activity for 5 minutes, and it comes back up if someone makes a request.
